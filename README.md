# Inferno Blob Audio Plugin

Plays "You're not that guy pal" audio when killing Blobs in the Inferno.

## Features
- Detects death of Jal-Ak (Level 360 blob)
- Detscts death of Jal-AkRek (Child blobs)
- Plays random audio clips for main blobs
- Plays pop sound for child blobs

## Configuration
- Adjust volume (0-100%)
- Enable/Disable sound effects
